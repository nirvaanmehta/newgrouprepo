# plot_bar_charts

## Purpose
Create bar charts for categorical variables.

## When to use
Use when the goal is to compare counts or proportions across categories.

## Inputs
- `df`
- one or more categorical variables
- figure directory
- optional `top_k`

## Outputs
Saved bar chart figures

## Why it helps an agent
Bar charts are one of the clearest ways to summarize survey items, group labels, and category distributions.

## Caution
If there are too many categories, the plot can become crowded and less useful.
