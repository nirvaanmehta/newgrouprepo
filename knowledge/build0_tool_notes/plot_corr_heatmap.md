# plot_corr_heatmap

## Purpose
Visualize a correlation matrix as a heatmap.

## When to use
Use when there are enough numeric variables that a table becomes difficult to scan.

## Inputs
- a correlation matrix
- output path

## Outputs
A saved correlation heatmap

## Why it helps an agent
It helps the agent detect clusters of related variables and communicate pairwise associations more clearly.

## Caution
A heatmap is useful for scanning patterns, but precise correlation values may still need to be reported separately.
